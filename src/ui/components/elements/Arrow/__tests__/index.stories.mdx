# Arrow

An icon component of an arrow, whose direction is determined by the value of the `direction` prop. This means that all arrow components are unified into one component. Furthermore the component also animates the change from one direction to another if the `direction` prop changes.

### Change

Upon pressing the 'Change' button the first arrow should rotate 90 degrees clockwise (while the second arrow should move counter-clockwise). Make sure that arrow incrementally rotates to the new position (animation).

_NOTE: After completing an entire the entire cycle (360 degress), the arrow will spin all the way back in the opposite direction to 0 degrees._
