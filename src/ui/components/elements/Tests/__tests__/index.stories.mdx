# Tests

**A React component used inside the project documentation to display the state of a specific component's tests.**

It only accepts a single prop, called `getData`. The `getTestData` method from the `src/ui/helpers/storybook` file is usually imported and called (i.e. what is returned from this function call is passed as a prop) if you want to use the results of a Jest test in here. Alternatively, you can pass a custom function that resolves an array of tests (as long as they fit the required scheme in the API below).

_NOTE: This component is intendedto only be used in `.mdx` files. If you are using it outside of an `index.stories.mdx` file then you are most probably using it incorrectly._
